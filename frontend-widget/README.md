# Solana Payment Widget

This folder contains a browser-only React widget intended to live at the repository root as a reusable package.

## Commands

```bash
npm install
npm run build
```

The build output is written to:

```text
examples/demo_project/solana_payments/static/solana_payments/solana-payment-widget/
```

That allows Django templates to serve the widget through the normal static files flow.
